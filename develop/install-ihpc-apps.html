Install Other Interactive Apps¶

For Jupyter, we provide a tutorial for copying a generic Jupyter batch connect template and modifying it to work at your center. See Add a Jupyter App to install Jupyter.

Table 6 Generic Interactive App Templates¶
Name GitHub URL
Jupyter https://github.com/OSC/bc_example_jupyter

While we don’t yet provide this for other interactive apps, here is a list of interactive apps that are currently deployed at OSC and the corresponding OSC templates.
