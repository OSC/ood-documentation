Overview¶

Configuring Open OnDemand to work with an Apache authentication module can be broken down into three procedures:

  1. Install and configure the Apache authentication module
  2. Setup the mapping of the remote user to the local user
  3. Configure logout

These are discussed in more detail in the following sections.
