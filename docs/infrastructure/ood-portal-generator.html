Generates an Apache configuration file that can be used for hosting an Open OnDemand portal on the OnDemand Server.

To get started generating an Apache configuration file you will need the following prerequisites:

  • The Ruby language version 1.9 or newer
  • The Ruby Rake software

Please see Software Requirements on how to install the above requirements.
