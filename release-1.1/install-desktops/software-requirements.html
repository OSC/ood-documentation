1. Software RequirementsÂ¶

The interactive Desktop app requires the following software to be installed on the nodes that the batch job is meant to run on, NOT the OnDemand node.

One of the following desktops:

  • Gnome 2 (currently we do not support Gnome 3)
  • Mate Desktop 1+

For VNC server support:

  • TurboVNC 2.1+
  • websockify 0.8.0+

Warning

Do NOT install the above software on the OnDemand node. The above software requirements are ONLY for the compute nodes you intend on launching Desktops on within batch jobs.
